> Script collection written in Deno.

## Usage

1. Install Deno
    - https://deno.land/#installation
    - https://x.deno.js.cn/#%E5%AE%89%E8%A3%85%E6%9C%80%E6%96%B0%E7%89%88

2. `npm add @cloydlau/scripts -g`

3. `cl -h`
